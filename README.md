**🖌️ Basic Paints** – Aplicación multiplataforma en desarrollo con **Flutter** y **SQLite**, orientada a coleccionistas y aficionados a las miniaturas.

🔧 Funcionalidades destacadas:
- Inventario personal de pinturas y miniaturas  
- Comparación visual de colores entre gamas comerciales  
- Registro de notas personalizadas por proyecto  
- Gestión offline, sin necesidad de conexión  
- Interfaz optimizada para Android (versión iOS en el futuro)

🚧 Actualmente en desarrollo y pruebas internas.
